# Laravel React Demo System

-   PHP 7
-   MySQL/Postgres
-   Laravel 7
-   Node.js
-   TypeScript
-   React
-   Material-UI
-   Vagrant

## Development

### First Run

```
git clone git@github.com:ecom/laravel-react-demo.git
cd laravel-react-demo
composer install
vagrant up
php artisan migrate
php artisan serve
```

### Daily Run

```
vagrant up
php artisan serve
```
